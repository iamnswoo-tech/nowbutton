# YoungButton v24.0 — 대시보드 통합 완료 패키지

## ✅ 이미 통합된 것 (이 패키지에 포함)
- `api/beacon.js` — 익명 통계 수신/집계 API (Serverless Function)
- `admin/dashboard.html` — 관리자 대시보드 (`/admin` 접속)
- `app.js` — Analytics SDK 내장 + _trackEvent 비콘 연동 (옵트아웃 포함)
- `index.html` — 보안 카드에 익명 통계 안내 + 끄기 토글 추가
- `vercel.json` — API 라우팅 + CSP + /admin rewrite
- `sw.js` — api/admin 경로는 캐시 우회

## ⚙️ 배포 전 딱 2가지만 교체하세요

### 1. 대시보드 URL 설정 (app.js 최상단)
```js
const YB_BEACON_URL = 'https://YOUR-DASHBOARD.vercel.app/api/beacon';
```
→ 실제 배포 도메인으로 교체. (교체 전엔 비콘이 전송되지 않아 안전합니다)

### 2. vercel.json CSP의 대시보드 URL
```
connect-src ... https://YOUR-DASHBOARD.vercel.app;
```
→ 같은 도메인으로 교체. (앱과 API가 같은 도메인이면 'self'로도 충분)

## 🚀 배포 순서
1. 이 폴더 전체를 GitHub에 push → Vercel 자동 배포
2. Vercel → Settings → Environment Variables:
   - `ADMIN_TOKEN` = (원하는 비밀 토큰)
3. (선택) Vercel → Storage → KV 생성 후 연결 → 데이터 영구 저장
   - KV 없으면 메모리 모드(서버 재시작 시 초기화)로도 동작
4. `https://앱도메인/admin` 접속 → 비밀번호 `youngbutton2026`

## 🔒 개인정보 보호 (중요)
- 수집: 측정 종류 + 점수 + 익명 세션ID만 (원시 측정값·이름·위치 없음)
- 사용자가 홈 보안 카드에서 **익명 통계 수집을 끌 수 있음** (옵트아웃)
- 처리방침 모달 문구도 정직하게 업데이트됨
- 끄면 비콘이 전송되지 않음

## ⚠️ 같은 도메인 vs 별도 도메인
- **같은 도메인 배포(권장)**: 앱과 api/beacon이 같은 Vercel 프로젝트 → CSP를 'self'로,
  YB_BEACON_URL을 '/api/beacon' 상대경로로 하면 가장 간단
- **별도 대시보드 도메인**: 위 2곳에 그 도메인 명시
